# NLC Demo Script

Target length: 2-4 minutes.

## Route

1. Home -> Search
   - Open the homepage and point out the search-first hero: "Search first. Recover securely."
   - Search for a common item such as "calculator" or "AirPods".
   - Mention public-safe item cards, clear statuses, and backend-unavailable/error/empty states.

2. Event Hub -> QR Beacon -> Report
   - Open `/EventHub` and choose the active PVHS demo event.
   - Show approved event-related Found Items and redaction note.
   - Open a zone/Beacon link and point out "You are reporting from: [zone]".
   - Click Report Lost Here or Report Found Here; show event/zone prefill is editable and no GPS is claimed.

3. Claim -> Admin Review -> Pickup
   - Open an item detail and submit/view a claim with identifying details.
   - Switch to Admin Dashboard and show Claims Queue permission gating.
   - Approve a claim, issue or view a Return Pass, then use Pickup Station to verify/redeem the pass.
   - Emphasize that students cannot mark a claim returned themselves and passes cannot redeem twice.

4. Recovery Center -> Sentinel -> Demo Builder
   - Show Recovery Center counts sourced from persisted records.
   - Show Loss Sentinel pattern review backed by Lost Reports.
   - In Demo Builder, create "Approved Calculator Return" or "Gym Electronics Pattern".
   - Explain cleanup requires `DELETE DEMO DATA` and deletes demo-flagged records only.

## Close

"Lost Then Found is not just a listing site. It is a complete, privacy-conscious recovery workflow: search, report, claim review, verified pickup, event context, pattern detection, and safe demo data management."
